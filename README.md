## VR Experiences
My course journey with bhanuprakash.<br>
Creating VR Experiences with React 360, Self-Study In CGVR.

### Install and run
```
1.  npm install
2.  npm start
3.  open "http://localhost:8081/index.html"
```

### Build
```
1. npm run bundle
2. copy build files + static_assets to server
```
